# Github helper scripts

## Installation

Note: you'll need a Python3.6.* installation.  

- `python -m venv venv`
- `source ./venv/bin/activate`
- `pip install -r requirements.txt`
- `export GH_USER=<your_user_name>`
- `export GH_SECRET=<your_secret_token>`
